# Lucky Throats Website

Official website source for Lucky Throats Barbershop.

## Stack

- Vite
- Vanilla JavaScript
- Sass (SCSS, BEM-style blocks)

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

Build output is generated to `dist/`.

## Project structure

- `src/` application source files
- `src/sass/` styles (base + block files)
- `src/js/` scripts
- `src/data/` content and translations
- `src/public/` static public assets
